# LEVEL 1 ACTION & BONUS ACTION GUIDE
# Beginner-Friendly Character System

---

# CORE DESIGN PRINCIPLES

## Level 1 characters should feel:
- simple
- readable
- thematic
- beginner-friendly

NOT:
- overloaded
- MMO-like
- ability-spam focused

---

# LEVEL 1 ACTION ECONOMY

Every character should usually have:

| Type | Amount |
|---|---|
| Action | 1 |
| Movement | 1 |
| Bonus Actions | 0–1 usually |

---

# GLOBAL ACTIONS
# Available To All Characters

| Action | Description |
|---|---|
| Attack | Perform weapon or unarmed attack |
| Cast Spell | Cast available spell |
| Dash | Gain extra movement |
| Dodge | Focus fully on defense |
| Hide | Attempt stealth |
| Search | Observe surroundings |
| Help | Assist ally |
| Disengage | Move away safely |
| Use Object | Interact with item or environment |

---

# GENERIC BONUS ACTION RULES

## Bonus actions should:
- feel meaningful
- be limited
- reinforce class fantasy

## Avoid:
- stacking many bonus actions
- free movement spam
- unnecessary complexity

---

# SAFE GENERIC BONUS ACTIONS

| Bonus Action | Requirement | Description |
|---|---|---|
| Off-Hand Attack | Dual wielding | Strike with secondary light weapon |
| Drink Potion | Optional system feature | Quickly consume potion |
| Spell Focus | Spellcaster only | Briefly stabilize magical concentration |
| Combat Focus | Martial classes | Enter disciplined battle stance |
| Silent Movement | Stealth-focused builds | Move quietly and carefully |

---

# BONUS ACTION GUIDELINES

# MARTIAL CLASSES

Usually receive:
- 1 combat-focused bonus action

Examples:
- Rage
- Second Wind
- Martial Arts

Avoid:
- excessive mobility chains
- repeated free attacks

---

# SPELLCASTERS

Usually receive:
- few or no bonus actions initially

Most spellcasters should focus on:
- actions
- positioning
- spell choice

NOT:
- bonus action spam

---

# STEALTH CLASSES

Should feel:
- precise
- agile
- tactical

BUT:
advanced mobility should unlock later.

Example:
- Rogue DOES NOT get Cunning Action at level 1.

---

# RACE FEATURE GUIDELINES

Most races should primarily grant:
- passive bonuses
- utility features
- lore flavor
- resistances

NOT:
- multiple combat actions

---

# SAFE RACE ACTION DESIGN

# GOOD

| Feature Type | Example |
|---|---|
| Passive resistance | Fire resistance |
| Vision enhancement | Darkvision |
| Minor utility | Mimicry |
| Limited action | Breath Weapon |

---

# BAD

| Problem | Example |
|---|---|
| Too many active abilities | 4+ combat buttons |
| MMO-style rotations | Combo chains |
| Early mobility overload | Free stealth dash spam |

# SIMPLE BUILDING RULES

# GOOD LEVEL 1 CHARACTER

Usually has:
- 1–2 core combat mechanics
- 0–1 meaningful bonus actions
- simple readable identity

---

# BAD LEVEL 1 CHARACTER

Usually has:
- too many abilities
- stacked mobility
- complicated resource systems
- excessive subclass mechanics

---

# SAFE LEVEL 1 IMPLEMENTATION STRATEGY

If uncertain:
DO NOT add the ability yet.

It is better for level 1 characters to feel:
- slightly simpler
than:
- mechanically bloated.

---

# RECOMMENDED LEVEL 1 EXPERIENCE

The player should immediately understand:
- what their class fantasy is
- what their main role is
- what their core combat loop is

WITHOUT:
- needing system mastery.

# GLOBAL COMBAT FORMULAS

## Attack Bonus

```text
Relevant Ability Modifier + Proficiency Bonus
```

### Typical Level 1 Range
```text
+4 to +6
```

---

# Damage Formula

```text
Weapon Dice + Relevant Ability Modifier
```

### Typical Level 1 Range
```text
1d4 + 2
to
1d12 + 3
```

---

# Sneak Attack

```text
+1d6 extra damage once per turn
```

Available ONLY to:
- Rogue

---

# SPELL ATTACK BONUS

```text
Spellcasting Modifier + Proficiency Bonus
```

---

# SPELL SAVE DC

```text
8 + Proficiency Bonus + Spellcasting Modifier
```

# LEVEL 1 CHARACTER POWER GUIDELINES

## Strong Melee Classes
Usually:
- +5 to hit
- 1d8 to 1d12 damage
- high AC

---

## Agile Classes
Usually:
- +5 to hit
- moderate damage
- stealth or mobility utility

---

## Spellcasters
Usually:
- lower weapon damage
- stronger utility
- ranged spell attacks
- lower AC

---

# IMPORTANT IMPLEMENTATION RULE

If a character:
- has too many bonus actions
- too many abilities
- too many attacks
- too much mobility

then the build is probably overloaded for level 1.

Level 1 should feel:
- readable
- thematic
- simple
- approachable.

# Action & Bonus Action System
# Levels 1–10

---

# GLOBAL ACTIONS

| Action | Type | Description |
|---|---|---|
| Attack | Action | Perform a weapon or unarmed attack |
| Cast Spell | Action | Cast a standard spell |
| Dash | Action | Gain extra movement this turn |
| Dodge | Action | Focus entirely on defense |
| Hide | Action | Attempt to remain unseen |
| Search | Action | Investigate surroundings |
| Help | Action | Assist an ally |
| Disengage | Action | Move without provoking attacks |
| Ready Action | Action | Prepare an action for later |
| Use Object | Action | Interact with an item or object |

---

# GLOBAL BONUS ACTIONS

| Bonus Action | Description |
|---|---|
| Off-Hand Attack | Strike using secondary weapon |
| Drink Potion | Quickly consume a potion |
| Rally Ally | Encourage nearby teammate |
| Combat Focus | Enter tactical combat stance |
| Spell Focus | Enhance magical concentration |
